# ✨ Overview

**Finora** is a modern offline finance dashboard designed to help users manage daily expenses, monitor budgets, and analyze spending habits through a smooth and elegant user experience.

The application focuses on:
- simplicity
- usability
- clean fintech-inspired design
- practical functionality

while remaining:
- fully offline
- lightweight
- backend-free

Everything runs directly inside the browser using a single HTML file.
